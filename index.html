Cette démo est un prototype non médical. Elle sert uniquement à tester une idée d’interface et de motivation.

Réglages parent / orthophoniste

Ces réglages simulent les adaptations possibles selon le profil sensoriel et le niveau de communication de l’enfant.

Pour certains enfants avec TSA, les animations motivent. Pour d’autres, elles peuvent surcharger. Le mode calme permet de réduire la stimulation.
